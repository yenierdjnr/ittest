# Accessibility

In a mission to make ITProTV IT learning content available to all aspiring and existing Office professionals, our development team has worked to improve the accessibility of our website. OfficeProTV is WCAG 2.0 AA compliant. If you have any suggestions or concerns regarding our accessibility, please email support@officepro.tv.

## New Features

- New accessibilities controls for primary and audio players
- New accessibilities controls for live video player
- Restructured course library tabbing
- Restructured ratings
- Restructured downloads and added shortcut to download button
- Added tabbing shortcuts to content, to video player, and to top of navigation
- Labeled carousel buttons
- Added language to main html tag
- Added roles to navigation menu
